Post-Processing: Temporal Consistency Filter

For each tile, scores across all consecutive image pairs (T1→T2, T2→T3, T3→T4) are evaluated together:

IF n_pairs_above_threshold ≥ 2 (MAJORITY_THRESH)
THEN all_scores × 0.6 (SEASONAL_DAMPEN)
# flags as seasonal drift, not true encroachment
✓ True Positive tile
Only 1 pair (neg→pos) flags
Dampening NOT triggered
Score stays high → TP
⚠ Seasonal FP tile
2-3 pairs all flag positive
Dampening triggered × 0.6
Score drops below thresh → TN

The t1_is_pos feature (#1 importance, 0.077) handles pos→pos pairs upstream — the temporal filter then catches remaining seasonal drift in neg→neg tiles.

Key Findings

✅ t1_is_pos is the #1 feature

Adding a single binary prior-label feature (was T1 already encroached?) improved Test AUC from 0.747 → 0.907 in one step. It teaches the model that pos→pos transitions are not encroachment events — the most common source of false positives in the v3 model.

🕒 Temporal consistency beats calibration

Probability calibration (Platt sigmoid, Isotonic) both degraded performance with only 6 val positives — the sample is too small to fit a reliable mapping. The temporal majority filter is a simpler and more effective post-processing step: it uses domain knowledge rather than statistical fitting.

📊 MNDWI dominates spectral features

MNDWI (Modified Normalised Difference Water Index) ranks 2nd–4th in importance. In Egypt's agricultural zones, encroachment correlates strongly with water channel proximity and irrigation-area changes — MNDWI captures this signal. BSI and NDBI rise as expected when built-up area expands.
