Methods. RF Classifier uses 44 tabular spectral/temporal features extracted per tile and a calibrated Random Forest (val AUC 0.91) — it answers "is this tile encroached?" Full Pipeline (Steps 01–07, fallback mode) runs spatial analysis on each pixel: Step 05 mean-abs-difference change detection, Step 06 NDVI > 0.2 agriculture mask, Step 07 NDBI_delta > 0.05 morphological building detection — it answers "where on the tile did new built-up appear on former farmland?"

Divergence (site105). The RF detects encroachment but Step 07 finds no building pixels. Likely cause: the encroachment is bare-soil conversion or early-stage land clearing without a detectable NDBI increase yet. The RF's broader 44-feature space captures this; NDBI-delta is sensitive only to confirmed built-up structures.

Fallback note. No GPU, ChangeFormer weights, YOLO weights, or SegFormer were used. All steps used rule-based fallbacks.
